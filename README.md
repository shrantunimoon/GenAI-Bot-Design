# GenAI-Bot-Design
# 🤖 GenAI Telegram Bot (Mini-RAG)

## 🚀 Features
- /ask → answers questions using documents
- Uses embeddings + LLM

## 🧠 Tech Stack
- Python
- sentence-transformers
- SQLite
- Ollama (LLaMA3)

## ⚙️ Setup

pip install -r requirements.txt

Create .env file:
TELEGRAM_TOKEN=your_token

Run:
python rag/embed.py
python app.py

## 🧪 Example

/ask How many leaves do employees get?

/ask Can I work from home daily?

/ask What happens if I share my password?
